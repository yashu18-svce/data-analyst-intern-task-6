# Sales Trend Analysis Using Aggregations

## Project Overview

This SQL project analyzes monthly revenue and order volume from an
`online_sales` dataset.

## Files

-   `task6.sql` -- Executable SQL script.
-   `online_sales.csv` -- Sample dataset (100 records).
-   `mysql_output_screenshot.png` -- Sample query output.

## Tools

-   MySQL Workbench / PostgreSQL / SQLite

## Concepts Used

-   GROUP BY
-   ORDER BY
-   SUM()
-   COUNT(DISTINCT)
-   YEAR()
-   MONTH()

## How to Run

1.  Create the database and table using `task6.sql`.
2.  Import `online_sales.csv`.
3.  Execute the SQL queries.
4.  Verify results with the sample screenshot.
